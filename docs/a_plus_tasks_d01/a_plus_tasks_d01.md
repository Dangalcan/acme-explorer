2. ~~Implement UI animations for the login, register, or master page interactions using Angular’s animation module.~~ DONE: Three CSS keyframe animations (`slideUp`, `fadeIn`, `shake`) are defined in `styles.scss` and exposed as Tailwind utility classes (`animate-slide-up`, `animate-fade-in`, `animate-shake`). The login and register cards slide up on load, the title fades in, error messages fade in, and the card plays a shake animation on failed login/registration attempts. The shake is driven by an Angular signal (`isShaking`) toggled in the component class, and reset on the `animationend` DOM event.
